## ⌨️ How to run this project ⌨️

### Requirements

- Node.js
- npm or yarn
- (Optional) API keys for any external AI services - This project is using Gemini API

### Execution

1. Clone this repository:

   ```sh
   git clone https://github.com/ventura-v/upload-ai-nlw-rocketseat.git
   cd upload-ai-nlw-rocketseat
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Configure environment variables in `.env` if required (for example external AI APIs).

   ```sh
   GEMINI_API_KEY="YOUR_API_KEY"
   ```

4. Start the dev server:

   ```sh
   npm run dev
   ```

5. Open the URL printed in the terminal (usually `http://localhost:5173` or similar).
